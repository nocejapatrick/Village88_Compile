@media queries

CSS users the media rule to detect information about the device being used to access the ewbsite. For example, it can be used to detect the size of the device. its capabilites, and its orientation. For this course. we will be using it to detect the device size, but keep in mind that it is capable of much more

Desktop: Width greater than or equal to 1024px

Tablet: Width between 481px and 1023px

Phone: Width less than or equal to 480px
